# WiFi Signal Monitoring System

## Overview
An IoT-based system that monitors WiFi signal strength (RSSI) in real time using ESP32 microcontrollers.

## Features
- Real-time RSSI monitoring
- MQTT communication using HiveMQ Cloud
- Node-RED dashboard visualization
- LCD display for live signal strength
- Packet Tracer network simulation

## Technologies Used
- ESP32
- Arduino IDE
- MQTT
- HiveMQ Cloud
- Node-RED
- LCD 16x2 (I2C)
- Cisco Packet Tracer
